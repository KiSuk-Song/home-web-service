## Dev-Ops 
- Docker (Completed Dec 2020)
    - Building docker image: `docker build -t flask-scheduler:latest .`
    - Running Container: `docker run --name flask-scheduler -d -p 7200:5000 --restart unless-stopped flask-scheduler`
